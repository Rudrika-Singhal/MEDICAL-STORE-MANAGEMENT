# MEDICAL-STORE-MANAGEMENT
A user-friendly application designed to streamline and automate the operations of a medical store. This system helps in managing medicines, tracking stock, handling sales and billing, and maintaining supplier information efficiently.

## Features

- **Inventory Management:** Add, update, and delete medicine records.
- **Stock Tracking:** Monitor medicine availability and expiry dates.
- **Sales & Billing:** Generate invoices for customer purchases.
- **Supplier Management:** Maintain supplier details and orders.
- **Reports:** View sales reports, stock reports, and purchase history.
- **User Authentication:** Secure login for store staff/admin.
